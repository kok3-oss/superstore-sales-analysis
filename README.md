# Superstore Sales Analysis

## Project Overview

This project analyzes Superstore sales data using Python to identify sales trends, regional performance, and top-performing products. The analysis includes data cleaning, KPI calculation, and visualizations to support business decision-making.

## Dataset

* Superstore Sales Dataset
* Records include orders, sales, customers, products, categories, and regions.

## Objectives

* Analyze overall sales performance
* Identify top and bottom performing products
* Compare sales across regions and categories
* Discover monthly and yearly sales trends
* Generate business recommendations

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Insights

* Technology category generated the highest sales.
* Certain regions consistently outperformed others.
* Sales showed seasonal patterns throughout the year.
* A small number of products contributed significantly to total revenue.

## Business Recommendations

1. Increase inventory for top-selling products.
2. Focus marketing efforts on high-performing regions.
3. Improve promotions during low-sales periods.
4. Review underperforming product categories.
5. Use customer segmentation for targeted campaigns.
